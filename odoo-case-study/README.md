## 🚀 Featured Project: Odoo ERP Aerospace Case Study
**[View Live Case Study](https://Roy-Chambers-GB.github.io/Professional-Portfolio/odoo-case-study/)**

### Overview
A comprehensive technical breakdown of a **Make-to-Order (MTO)** environment. This implementation focuses on the "Digital Thread," ensuring full traceability from initial customer demand through to final delivery.

### Key Technical Features:
* **End-to-End MTO Architecture:** Integrated Sales, Manufacturing (MES), and Logistics.
* **Compliance & Traceability:** Automated NDT (Non-Destructive Testing) certification checkpoints.
* **Shop Floor Control:** Real-time production visibility and serialised component tracking.
* **Data Integrity:** Multi-stage routing with integrated Bill of Materials (BoM) accuracy.
